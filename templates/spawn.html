Get Started

Piksel Sandbox adalah Jupyter Notebook berbasis komputasi-awan yang didesain untuk belajar dan bereksperimen dengan dataset dan library geospatial.

Apa yang bisa Anda temukan di sini:
  • Geospasial Data-Science Toolkit

    Sandbox ini dilengkapi dengan berbagai tools penting untuk data science, seperti geopandas dan library lainnya untuk analisis data geospasial

  • Contoh untuk Belajar

    Jelajahi berbagai contoh dan pelajari cara kerja dengan data geospasial.

    • Mulai dari hal dasar, seperti loading, analisis, dan plotting data observasi bumi menggunakan Datacube.
    • Eksplorasi aplikasi praktis, seperti mapping garis pantai.
  • Resource yang Dijamin

    Setiap user mendapat resource khusus untuk mendukung pekerjaan mereka, mulai dari 2 vCPU dan 16GB RAM.
